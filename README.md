# Barley-Vintage - 15 puzzle for iOS using Swift and SpriteKit.
The 15 puzzle consists of 15 squares numbered from 1 to 15 that are placed in a 4Ã—4 box leaving one position out of the 16 empty. The goal is to reposition the squares from a given arbitrary starting arrangement by sliding them one at a time into the configuration shown above. For some initial arrangements, this rearrangement is possible, but for others, it is not.
